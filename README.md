# XDesign

一套专为 React 开发，适合开发者使用的轻量级 UI 组件库

## 特点

- 专为 React 开发的 UI 组件库
- 组件体积更小，灵活使用，按需使用
- 极少的代码量，极简配置项

## 安装

使用 npm 的方式安装

```npm
npm i xdesign-ui
```

## 按需引入

您可以根据个人需要按需引用组件来使用

```tsx | pure
import { Button } from 'xdesign-ui';
```

## 完整示例

```tsx | pure
import React from 'react';
import { Button } from 'xdesign-ui';

const Index = () => {
  return <Button>按钮</Button>;
};

export default Index;
```
