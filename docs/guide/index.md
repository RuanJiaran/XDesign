title: 快速开始
---

## 安装

使用 npm 的方式安装

```npm
npm i xdesign-ui
```

## 按需引入

您可以根据个人需要按需引用组件来使用

```tsx | pure
import { Button } from 'xdesign-ui';
```

## 完整示例

```tsx | pure
import React from 'react';
import { Button } from 'xdesign-ui';

const Index = () => {
  return <Button>按钮</Button>;
};

export default Index;
```

更多使用示例请看 [组件文档](/components)
